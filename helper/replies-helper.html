What this does

This tool talks to Ghost through a small PHP proxy (so the Admin API key stays server-side). When you save, it does three things:

1. Fetches the post's current codeinjection_foot field.
2. Merges new valid URLs with any already-saved URLs (no duplicates).
3. Writes the updated JSON block back to Ghost.

The change is live on the blog immediately — no template updates needed.
